# LetsGetAJob 🚀

Your personal job hunt HQ. An all-in-one AI-powered career command center designed to turn the chaos of job hunting into a structured, high-conversion pipeline.

## Features

- **AI Fit Check**: Instantly score your profile against any job description.
- **Resume Studio**: Tailor your CV bullets and draft cover letters using AI.
- **Smart Pipeline**: Visual dashboard with "Wall of Shame" and priority alerts.
- **Network Intelligence**: Track high-value connections and follow-ups.
- **CV Library**: Store and manage multiple tailored versions of your resume.

## Tech Stack

- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Lucide Icons**
- **TypeScript**

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Deployment

This project is configured for one-click deployment on **Vercel** or **Netlify**.
